# 브라질 이커머스 (Olist) 탐색적 데이터 분석(EDA) 리포트
해당 분석은 9개의 Olist 관련 데이터셋을 통합하여 분석한 결과입니다. Olist는 브라질 이커머스 쇼핑몰의 통합 판매망 데이터를 나타냅니다.
